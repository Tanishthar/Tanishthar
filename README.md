## ⚡ About Me

I am a developer building at the intersection of **operating systems**, **agentic AI architectures**, and **deep learning**. I design systems that own the entire stack—from low-level kernel routines to multi-agent LLM loops and 3D computer vision models.

* 🚀 **Currently Building:** Multi-agent coordination systems, C++ trading architectures, and medical diagnostic pipelines.
* 🛠️ **Systems Focus:** 64-bit OS kernel design (Ring 3 isolation, scheduler, TCP/IP stack), low-latency networking, and low-level optimization.
* 🧠 **AI Focus:** Multi-Agent Reinforcement Learning (MARL), explainable 3D computer vision (Grad-CAM), and self-verifying agent playbooks.
* 💡 **Philosophy:** Build the foundations first. Write the kernel, implement the protocol, understand the weights, and ship code that lasts.

## 🚀 Featured Engineering Projects

### 🌌 [TanOS](https://tanishthar.netlify.app/projects/tanos) | `C` `x86-64 ASM` `Systems`
*From a custom UEFI bootloader to a graphical windowed desktop that browses the real web over its own TLS stack — built entirely in C from scratch.*
* 📑 **[Read Case Study & View Demo](https://tanishthar.netlify.app/projects/tanos)**
* **Rigorous Specs:** 50,000+ lines of hand-written C/ASM | 49 Ring 3 syscall ABI | 17 apps (graphical App Store, File Manager) | e1000 network driver.
* **Key Achievement:** Implemented *Compass*, a custom-written web browser layout engine rendering HTML, a CSS subset, PNG images, and forms over validated TLS certificates (using 123 Mozilla CA anchors).

### 🤖 [Issue Triage Orchestrator](https://tanishthar.netlify.app/projects/issue_triage) | `LangGraph` `Docker` `FastAPI`
*A multi-agent production software issue triaging system powered by LangGraph, Docker sandboxes, and PGVector.*
* 📑 **[Read Case Study & View Demo](https://tanishthar.netlify.app/projects/issue_triage)**
* **Rigorous Specs:** 115+ passing backend tests | 3 collaborating agents with tool-calling flows | Sandboxed code execution with automated retries and circuit-breakers.
* **Key Achievement:** Built a real-time tracking dashboard in Next.js showing live step-by-step reasoning logs, tracking metrics (e.g. `tool_error_rate`, `eval_score`) saved directly in `metrics.json`.

### 🧠 [Alzheimer 3D-MRI Diagnostic AI](https://tanishthar.netlify.app/projects/alzheimer-detection) | `PyTorch` `3D CNN` `Medical AI`
*A clinically grounded, explainable deep-learning system for stage-wise Alzheimer detection using raw 3D MRI brain scans.*
* 📑 **[Read Case Study & View Demo](https://tanishthar.netlify.app/projects/alzheimer-detection)**
* **Rigorous Specs:** Wrangled 12 disks of OASIS-3 data | 154 subjects segmented | ~4 min per 3D scan on WSL | Grad-CAM clinical explainability.
* **Key Achievement:** Developed an end-to-end preprocessing, anatomical segmentation (isolating hippocampus and brainstem), and stage-wise classifier pipeline that maps predictions back to brain regions responsible for clinical trust.

### 🎮 [AI Multi-Game RL & Arena Platform](https://tanishthar.netlify.app/projects/ai_game_agents) | `PyTorch` `Three.js` `Socket.IO`
*A full-stack reinforcement learning research platform featuring 7 algorithms, 6 games, and a 2v2 capture-the-flag arena.*
* 📑 **[Read Case Study & View Demo](https://tanishthar.netlify.app/projects/ai_game_agents)**
* **Rigorous Specs:** 2v2 Capture-the-Flag with MAPPO-lite (CTDE) | Real-time action probability & value function HUD overlays | Automated PDF research reports.
* **Key Achievement:** Watch cooperative agents think in 3D with spatial heatmaps, trajectory tracing, and frame-by-frame post-hoc match scrubbing.

### 📈 [QuantForge](https://tanishthar.netlify.app/projects/quantforge) | `C++` `React` `FastAPI` *(Ongoing)*
